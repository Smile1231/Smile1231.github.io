# YAML 如何书写

YAML 语法

支持的数据结构

基本语法

  • k: (空格) v： 表示一对键值对
  • v: 包含上面举的数据
  • k;string 类型

value 的使用说明

  • 基本类型：数字，字符串，布尔

    • k: v： 字面直接来写；
    • 字符串默认不用加上单引号或者双引号；

  • "" ：双引号：不会转义字符串里面的特殊字符；

    • 特殊字符会作为本身想表示的意思；
    • 如 ： name: "zhangsan \n lisi"： 输出； zhangsan 换行 lisi

  • '' ：单引号：转义特殊字符

    • 特殊字符终只是一个普通的字符串数据 ，
    • 如： name: ‘zhangsan \n lisi’：输出；zhangsan \n lisi

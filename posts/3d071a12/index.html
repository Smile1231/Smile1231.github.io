# 设置 page

点击 Settings ，向下拉到最后有个 GitHub Pages ，点击 Choose a theme 选择一个主题。然后等一会儿，此时点击图片中打码链接就是你的 GitHub Pages 地址了。应该是会跳出一个页面出来。
